# cmake-fetchcontent-example
example of a repo that's ready to use with `FetchContent_Declare`

This is the most basic project structure for a project that can be pulled in using `FetchContent_Declare`:

```
MyLibrary/
│
├── include/
│   └── MyLibrary/
│       └── hello.h
│
└── src/
    └── hello.cpp
└── CMakeLists.txt
```
